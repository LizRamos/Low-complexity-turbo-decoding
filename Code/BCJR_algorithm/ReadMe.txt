Ce dossier contient le code d'une implémentation de l'algorithme BCJR, et des tests pour mesurer ses performances en tant qu'algorithme de décodage de codes convolutifs.
Ce code doit servir de base à l'implémentation de turbo-codes en rendant possible l'emploi de code récursifs et la transmission d'informations extrinsèques en sortie du bloc de décodage BCJR.
